 # 映画トーク

## サイト概要

映画の紹介やユーザーが映画について話合うことができるサイト。

### サイトテーマ

映画の共有

### テーマを選んだ理由

映画鑑賞が趣味であり、映画を紹介したり感想を話し合えるようなサイトを制作してみたいと思いこのテーマしました。

### ターゲットユーザ

映画好きな人や映画の内容を共有したい人

### 主な利用シーン

気になる映画を調べたり、映画の感想を共有したいとき
